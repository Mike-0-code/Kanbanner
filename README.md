# Kanbanner - Tablero Kanban Simple

Organiza tus tareas con un tablero Kanban elegante y fácil de usar directamente desde tu navegador.

## Características

- Gestión visual de tareas con columnas Por Hacer, En Progreso y Hecho
- Arrastrar y soltar intuitivo entre columnas
- Persistencia automática en tu navegador
- Diseño completamente responsive
- Completamente privado - tus datos se quedan en tu dispositivo

## Cómo usar

- Crear tarea: Haz clic en "Agregar tarea" en cualquier columna
- Editar: Haz clic en cualquier tarea para modificar título y descripción
- Mover: Arrastra tareas entre columnas para seguir tu progreso
- Eliminar: Usa el botón "Eliminar" al editar una tarea
